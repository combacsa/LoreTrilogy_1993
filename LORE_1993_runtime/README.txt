1. 이번 버전에서 달라진점


2. 캐릭터 만들기


 1) 자신의 이름과 성별을 기입한다. ( 성별도 능력에 영향을 미친다.)

 2) 여러 질문에 대답을 하여 성격을 만들어낸다.

      체  력 : 캐릭터가 지니고 있는 본질의 힘. 무기의 공격력에 영향을 준다.

      정신력 : 마법지수에 영향을 미친다.

      집중력 : 초감각지수에 영향을 미친다.

      인내력 : 캐릭터의 생존력의 척도가 된다.

      저항력 : 적들의 육체적, 정신적 공격을 차단하는 능력

      민첩성 : 전투가 시작될때 먼저 알아차리고 공격하는 능력

      정확성 : 육체적, 정신적 공격의 정확성을 나타낸다.

      행  운 : 우발적인 피해를 줄이며, 적들의 특수공격을 피할수 있다.

      방어력 : 적에 의한 피해를 감소 시키는 역할을 한다.

      경험치 : 전투 경험을 나타낸다.

      레  벨 : 전투 경험에 따른 훈련으로 이 수치를 증가 시키며,  모든 행동
               에 가장 큰 영향을 미친다.

 3) 자신의 계급을 선택한다.

      기  사 : 여러가지 무기를 더 효율적으로 다룰수가 있으며,  방어력 또한
               다른 계급 보다 더 높다. 하지만, 마력과 초능력에는 재능이 전
               혀 없다.

      마법사 : 마법을 최대한 사용할수 있으며, 초능력 또한 중간 수준이다.
               하지만, 육체적인 공격과 방어에는 약하다.

      에스퍼 : 최고 레벨의 초능력과 중간급의 마법을 사용한다.  그렇지만 육
               체적인 능력은 약간 떨어진다.

      전  사 : 기사에 버금가는 공격을 벌일수가 있으며,  고수준의 마법을 펼
               친다. 어떤 공격과 방어에도 적합한 캐릭터이다.

      전투승 : 맨손으로만 공격을 펼치지만 그 기량은 기사 계급을 능가한다.
               하지만, 마법과 초능력에는 재능이 전혀없다.

      닌  자 : 전사급의 무기 공격에다 중급의 마법과 초능력을 사용한다.  저
               항력과 민첩성은 뛰어나지만 맨손 공격은 약하다.

      사냥꾼 : 모든 공격에 정확성이 있으며 다른 조건은 평균적이다. 하지만,
               마법과 초능력에는 재능이 전혀없다.

      떠돌이 : 이곳 저곳을 떠돌아 다니는 특정한 계급이 없는 캐릭터이다.

      혼  령 : 육체적인 힘은 없지만 마법의 힘이 뛰어나다.

      반  신 : 신과 인간 사이의 중간 존재이며 모든 능력이 다른 계급을 압도
               한다.

 4) 자신과 동행할 4 명의 캐릭터를 고른다.

       이때 선택한 인물이 반드시 마지막까지 있어야 한다는 보장은 없다.
       자신의 일행은 총 6 명까지이며 마지막 대원은 인간이 아닐 수도 있다.
       중간에 만나는 능력있는 인물들과 합류할 수도 있고 필요없는 인물을 제
      명 할 수도 있으며, 공격 순서를 바꿀 수도 있다. (단, 마지막 대원은 예
      외 이다.)

 5) 일행의 결성 여부를 선택 한다.

   2> 치료 마법
      : 이 마법에 대해서는 메뉴만으로도 이해가 될 것이다.

    ***  주의 : 모든 치료는 심도가 깊은 곳부터 해야 한다.

                   부상  ==>  독  ==>  의식 불명  ==>  죽음

                즉, 독이 퍼져있는 상태에서는  부상을 치료할 수가 없고 의식
               불명인 상태에서는 독에 대한 치료를 할 수가 없다.

                대부분의 죽을 경우에는 의식 불명의 상태를 거치지만, 그렇지
               않은 '즉사'의 기술을 가진 캐릭터 (Sprite, Medusa, Lord Ahn,
               Black Knight ... 그리고 Necromancer) 에 의한 죽음을  치료할
               때는 '부활' 기술로만 다시 살려낼 수 있다.

   3> 변화 마법
      : 괄호안은 사용 가능한 마법 레벨을 나타낸다.

      마법의 횃불 ( Lv1 ) : 가장 많이 쓰이며, 동굴속의 어둠을 제거 시킨다.
                  일행이 쉴때 불은 꺼지며 이것이 통하지 않는 지대도 있다.

      공중 부상 ( Lv2 ) : 모든 일행을 공중에 떠있게 한다. 하지만 그리 좋은
                  기술은 아니어서 아래의 지형이 불안정한 물과 늪 위는 지날
                  수가 없으며, 계곡 사이를 지나갈 때 쓰인다.

      물 위를 걸음 ( Lv4 ) : 물 위를 걸을수 있으며 횟수에 제한이 있다.  하
                  지만 깊고 거친 바다 위는 갈 수 없다.

      늪 위를 걸음 ( Lv6 ) : 위의 마법과 유사하다.

      기화 이동 ( Lv8 ) : 일행을 기체로 만들어 벽을 통과한 후  다시 원래대
                  로 되돌린다.  즉, 지나갈 수 없는 1 칸을 건너 뛰어 이동한
                  다.

      지형 변화 ( Lv10 ) : '기화 이동'과는 달리 지나갈 수 없는 지형을 흙으
                  로 변화시켜 버린다. 아주 유용한 마법이다.

      공간 이동 ( Lv12 ) : 최고 9 칸 까지의 지형을 건너 뛰어 이동한다.  이
                  게임에서 가장 유용한 마법이며  진행을 무난하게 만들어 준
                  다. 이 마법이 통하지 않는 지대도 있다.

      식량 제조 ( Lv14 ) : 주위의 물질을 이용해 약간의 식량을 만들어 낸다.

 3) 초능력

      투  시 : 현재의 지형에서 변화의 여부가 있는 곳을 꿰뚫어 본다.

      예  언 : 다음에 하게 될 일을 예언해 낸다.

      독  심 : 평상시에 사용하게 되면  잠시 남의 마음을 읽어 마음을 조종하
              여 자기 쪽으로 남을 끌어 들일수 있으며, 전투시에 사용하게 되
              면 적을 우리편으로 끌어 들일수 있게 된다. 하지만 전혀 우리쪽
              으로 오지 않는 적도 많다.

      천리안 : 초감각 레벨에 따라 먼곳을 내다 볼 수있다. '공간 이동'마법과
              함께 사용하면 대단히 효과적이다.

      염  력 : 이 것을 사용하면 상황에 맞게 약 10여가지의 변화 공격이 펼쳐
              지게 된다.


4. 게임 진행 방법

      사용 되는 키는  < 화살표 키 >, < Space >, < Enter >, < Esc > 가 전부
     이며 추가적으로 스피커의 소리를 토글 시키기 위해 < Back Space > 가 사
     용 된다.

      < 화살표 키 > : 캐릭터를 이동 시키거나 메뉴를 고를 때 사용한다.
                      이 키로 대화를 할 수 있다.

      < Space >     : 주 메뉴를 나타나게 한다.

      < Enter >     : 메뉴 선택 등에 사용된다.

      < Esc >       : 상점등의 대화에서 그전 대화로 돌아갈때 쓰인다.

      그 이외에 단축키가 있다.

        'p' : ( Party view ) 일행의 상태를 본다.
        'v' : ( View character ) 개인의 상태를 본다.
        'q' : ( Quick view ) 일행의 건강 상태를 본다. 독이 퍼진 정도, 의식
                            을 잃은 정도, 죽음의 정도 등등.
        'c' : ( Cast ) 마법을 사용한다.
        'e' : ( Esp using ) 초능력을 사용한다.
        'r' : ( Rest here ) 잠시 쉬며 의식을 회복하거나 상처를 치료 한다.
                           이 때 식량이 필요하며 독이 퍼진 상태에서는 치료
                           가 불가능 하다.
        'g' : ( Game option ) 게임에 대한 선택 상황을 조절한다.

             "난이도 조절"  : 적이 출현하는 빈도수와 최대 출현수를 조절.
             "순서를 정렬"  : 일행의 순서를 정렬. 6 번째 대원은 제외.
             "일행에서 제외": 자신을 제외한 모든 대원을 탈퇴 시킬 수 있다.
             "게임 재개"    : 저장 했던 게임을 불러 온다. '본 게임 데이타'
                             에 저장 되어 있는 것이 디폴트로 게임 시작시에
                             불려 진다.
             "게임 저장"    : 현재의 게임을 저장 한다.
             "게임을 마침"  : 다시 Dos 로 돌아 간다.
